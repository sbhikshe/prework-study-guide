HTML

  • The head element contains info about the webpage
  • The title tag text appears on top of the page in the browser
  • The other tags are meta, script, style, base, link

  • The body element contains content that is visible to the user
  • h1 - 6 are used for headings
  • br tag for line break
  • p - for wrap around paragraph
  • ul - unordered list, with li list items
  • ol - ordered list with li list items
  • img - image tag with src, alt, width and height attributes
  • a href for URLs
  • script - to link to a JavaScript file

CSS

  • CSS can be added to HTML inline, or internal / external stylesheets
  • inline CSS with the element - h1 style="color:aqua
  • internal: style element in the head portion
  • external: use the link element to apply an external .css file.
  • For example link to an css file rel = “stylesheet” href = “x.css”
  • Reference to CSS

GIT

  • git clone SSH link
  • git status - to check the current branch
  • git checkout main - to switch to the main branch
  • git checkout -b branchname - to create and switch to new branch
  • git commit
  • git pull / push

JavaScript

  • add to a .js file, and link to html file with script tag in the body
  • ; are optional, but good practice at the end of a line
  • Variables: var var_name
  • Variables can be reassigned to a value of a different type!
  • Arrays: var arr_name = [ , ] - arrays can be a mix of data types
  • Functions: function fn_name() - arguments, return values?
